Este é um projeto básico de java orientado a objetos feito em outubro de 2020 e que está sendo revisado para praticar a manutenção de códigos antigos.  A idéia era criar um sistema que possa simular o gerenciamento de clientes e de suas contas bancárias, em um banco fictício. 

Existem duas classes que podem ser executadas uma para verificar o funcionamento de funcoes e outra que seria a implementação do menu e ambas precisam de uma lista de clientes que será a fonte de dados.

Um único cliente terá uma lista de ContaBancaria e cada conta será acessada pelo seu código que vai ser incremental (o código de cada conta será a posição do seu vetor + 1, assim codConta-1 é a posição que a conta está no vetor conta do objeto Clientes).

O arquivo Decisao.java é uma classe operária que serve para encapsular linhas de comandos referentes à escolha do usuário no menu principal.
